# ZOME · 造

> AI + 你的照片 = 无限 IP 资产

ZOME 是一个从一张照片出发、造出完整 IP 角色资产矩阵的 Web 应用。上传人物、宠物、物品或品牌照片，AI 自动取色分析并生成专属卡通 IP 形象，再一键衍生出表情包、人设卡、节日海报、周边 Mockup、社交封面等全套视觉资产。

## 为什么做

市面上的 AI 生图工具止步于"生成一张头像"，缺乏后续延展。ZOME 要解决的是：让一张照片变成一套可以被喜欢、被分享、被持续运营的角色资产。

- 养宠人群想做表情包，但不会画画，现有工具千篇一律
- 个人想要专属形象，但 AI 工具只出一张图没有后续
- 小品牌想做吉祥物，请设计团队成本高、周期长

---

## 核心特性

**一张照片即可启动** — 无需填表，上传后自动取色与形象分析

**四种创作风格** — 可爱风 / 潮酷风 / 复古风 / 手绘风，一键切换

**六类 IP 资产** — 主形象、表情包套装、角色人设卡、节日海报、周边 Mockup、社交封面

**批量生成** — 确认形象后自动批量产出全套资产，无需逐张操作

**纯前端部署** — 基于 GitHub Pages，零后端依赖，开箱即用

---

## 使用流程

```
上传照片 → AI 自动取色分析 → 生成 IP 主形象 → 切换风格 / 调整姿势 → 确认生成 → 批量产出全套资产
```

1. **上传照片** — 支持人物、宠物、物品、店铺照片，也可点击示例直接体验
2. **AI 看图造 IP** — 自动提取照片主色调，分析形象特征，生成卡通 IP 主形象
3. **确认形象** — 切换创作风格，滑动调整姿势，满意后进入下一步
4. **批量造资产** — 自动生成表情包、人设卡、海报、周边 Mockup、社交封面等

## 快速开始

### 环境要求

- 现代浏览器（Chrome / Edge / Safari / Firefox 最新版）
- 如需生成 IP 资产，需配置图像生成模型 API Key

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/qwhe/project.git

# 进入项目目录
cd project/ZOME

# 直接用浏览器打开 index.html，或启动本地服务器
python3 -m http.server 8080
```

浏览器访问 `http://localhost:8080` 即可体验主页，体验入口在 `http://localhost:8080/demo1/`。

### 配置生图 API

在体验页中，生图模型默认显示"未连接"状态。如需启用 AI 生成功能，请在项目配置中填入你的图像生成模型 API Key。

## 开发指南
